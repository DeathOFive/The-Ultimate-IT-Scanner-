PlugNPlay IT Hardening and Security Audit SuiteAn interactive, cross-platform security auditor designed exclusively for Windows and Linux hosts. It allows network students, developers, and system administrators to audit loopback port configurations, inspect firewall policies, check anti-malware status, and selectively check for registered system vulnerabilities using the live NIST National Vulnerability Database (NVD).Key FeaturesExclusive OS Compatibility GuardrailSafely supports mainstream Windows and Linux environments. Operates with an automatic termination guardrail if run on macOS to prevent unintended behavior.Detailed OS Version DetectionAutomatically gathers precise operating system details, such as your specific Windows release or Linux distribution name and version.Interactive CVE Query GateOffers a privacy-conscious console prompt asking whether you want to query the live NVD API for vulnerability records explicitly matching your operating system version.Loopback Port ProfilingFast, multi-threaded localhost network audit scanning for standard services and administrative ports (1-1024).Automated Dependency SetupAutomatically detects and silently installs required third-party libraries (psutil and requests) on demand.Dual ReportingAutomatically generates a modern, dark-themed HTML visual report (scan_report.html) alongside a structured data dump (scan_results.json).Installation and Usage1. Ensure Python is InstalledMake sure you have Python 3 installed on your machine. You can verify this by opening a terminal or Command Prompt and running the following command:python --version
Note for Windows Users: During the Python installation process, ensure you check the box that says "Add Python to PATH".2. Running the AuditDownload the plugnplay.py script and run it from your command line interface.On WindowsOpen Command Prompt or PowerShell with Administrator privileges and run:python plugnplay.py
On LinuxOpen your terminal. It is recommended to run the script with elevated privileges to allow the parser to audit restricted system files (such as SSH configurations and sudoers):sudo python3 plugnplay.py
User Controls and Command Line OptionsInteractive Prompt: NVD CVE QueryAt startup, the tool will identify your specific OS version and present the following query:Audit OS-specific CVE records? (y/n):
Pressing 'y' (Yes): The tool securely connects to the live NIST NVD API to find up to 5 critical vulnerabilities associated with your specific OS version.Pressing 'n' (No): Bypasses internet-facing queries entirely. This is ideal for offline diagnostic runs, air-gapped systems, or preserving network bandwidth.Getting HelpYou can print the built-in user manual directly in your terminal at any time by passing the help flag:python plugnplay.py --help
Scan OutputsUpon a successful audit, the suite automatically writes two files to your current directory:scan_report.htmlA highly stylized, dark-themed HTML report containing responsive tables of open ports, categorized risk indicators, security configurations, and live threat streams.scan_results.jsonA raw, structured JSON file containing all diagnostic metrics. This is perfect for database archiving, spreadsheet imports, or custom automated scripting.TroubleshootingModuleNotFoundErrorThe script attempts to auto-install dependencies (psutil and requests). If your system's network policies block auto-installation, you can manually install them by running:pip install psutil requests
Permission Errors on LinuxIf the script is unable to check SSH configurations, make sure to execute the script using the superuser command: sudo python3 plugnplay.py.macOS Execution ErrorThe tool is designed specifically for Windows and Linux hosts. Execution on macOS is intentionally blocked at runtime to prevent configuration diagnostic mismatches.
